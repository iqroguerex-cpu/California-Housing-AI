# 🏠 California Housing Price Prediction

[![Streamlit](https://img.shields.io/badge/Built%20with-Streamlit-FF4B4B.svg)]()

A Machine Learning dashboard built with **Streamlit** that predicts California housing prices using **Linear Regression** on the Scikit-Learn dataset.

## ⚙️ Tech Stack

- Python  
- Scikit-Learn  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Streamlit
